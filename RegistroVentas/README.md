# Sistema de Registro de Ventas

## Descripción

Aplicación desarrollada en C# Windows Forms que permite registrar ventas de productos.

## Funcionalidades

- Registrar producto
- Registrar cantidad
- Registrar precio unitario
- Calcular subtotal
- Mostrar ventas en DataGridView
- Calcular Valor Venta
- Calcular IGV (18%)
- Calcular Total
- Validaciones de datos

## Tecnologías

- C#
- Windows Forms
- .NET Framework
- Visual Studio 2022
